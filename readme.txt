== Description ==

BuddyPress Cover It allows users to have a cover profile.

Allow user add upload avatar function with ajax in profile.

user can drag cover and save position. 

send activity when user change cover and avatar.

user can select from previously uploaded cover photos.

admin can upload default avatar , cover profile and cover group

== Installation ==
= To Install: =

1.  Download the plugin file
2.  Unzip the file into a folder on your hard drive
3.  Upload the `/bp-cover/` folder to the `/wp-content/plugins/` folder on your site
4.  Single-site BuddyPres go to Plugins menu and activate there.
5.  For Multisite visit Network Admin -> Plugins and Network Activate it there.
